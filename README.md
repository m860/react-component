# webpack-react-template
react webpack项目模板

持续更新...

### 安装
```bash
$ git clone https://github.com/m860/webpack-react-template.git
$ cd webpack-react-template
$ npm install
$ npm install -g webpack
```
### 开发模式
```bash
$ npm run dev
```
然后在浏览器中访问:http://localhost:3000

### 发布模式
```bash
$ npm run build
```
